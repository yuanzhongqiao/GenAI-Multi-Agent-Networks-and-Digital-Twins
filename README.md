<p dir="auto" _msttexthash="3546149477" _msthash="248">虽然有帮助，但除了 HS 和 Python 入门之外，没有其他数学是必需的。与传统的 CS/AI 课程不同，我们不是从综合理论开始自下而上构建。相反，我们遵循一种自上而下的方法，从高抽象层次的实际问题开始。我们为每个学生提供了一个知识框架，让他们根据自己的个人准备和观点来理解基本理论。</p>
